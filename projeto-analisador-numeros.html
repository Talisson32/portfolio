Resultado

Desenvolvido com HTML, CSS e JavaScript puro, este analisador de números permite que o usuário insira valores de 1 a 100 e obtenha estatísticas completas sobre os números digitados. Aplicação criada para melhorar a lógica.

  • Validação de Entrada - Aceita apenas números entre 1 e 100 com alertas de erro
  • Adição Dinâmica – Os números são adicionados e exibidos em tempo real
  • Análise Estatística – Calcula quantidade total, soma, média, menor e maior valor
  • Ordenação Automática – Os números são organizados em ordem crescente
  • Interface Responsiva – Design adaptável com gradientes e efeitos visuais
