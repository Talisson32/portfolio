Projeto acadêmico, desenvolvimento Desktop utilizando linguagem JAVA

Sistema Desktop JAVA de Gerenciamento de EPIs

Tecnologias Utilizadas

  • Java SE 8 - Backend
  • Swing – Componentes visuais desktop
  • PostgreSQL 16 – Banco de dados relacional .
Ver no GitHub

Desenvolvido em Java 8 (Java SE) utilizando NetBeans, sistema desktop oferece uma interface para login, cadastro de novos usuários, listagem de todos os EPis cadastrados, tela para cadastro de epis e uma tela para manutenção o modelos dos EPis. A interface gráfica foi construída usando o GUI Builder do NetBeans (Drag & Drop).

  • Login - Tela para realizar login e cadastro de novos usuários
  • Visualizar – Lista todos os EPis cadastrados.
  • Tela EPI – Tela de cadastro EPi com dropdown Categoria carregada do banco (CRUD).
  • Tela Categoria – Tela para gerenciar categorias (CRUD).
